# Template 03: Benefit Stack / Product Bundle

## Description
Three products stacked vertically, each paired with an emoji + benefit statement. Discount badge at top. Makes a multi-product offer feel like one cohesive system.

## Visual Layout

```
┌─────────────────────────────┐
│  🏷️ [DISCOUNT]% OFF         │
├─────────────────────────────┤
│  [BUNDLE NAME]              │
│  in bold headline           │
├─────────────────────────────┤
│                             │
│  [Product 1 img] 🧠 [Benefit 1] │
│  ─────────────────────      │
│  [Product 2 img] 🩸 [Benefit 2] │
│  ─────────────────────      │
│  [Product 3 img] 👁 [Benefit 3] │
│                             │
├─────────────────────────────┤
│  * [Fine print/disclaimer]  │
└─────────────────────────────┘
```

### Zones
- **Discount Badge (10%):** Colored badge/pill with percentage off. Top-left or top-center.
- **Headline Zone (10%):** Bold bundle name (e.g., "STAY SHARP BUNDLE")
- **Product Stack Zone (70%):** Three horizontal rows, each with product image (left) + emoji + benefit text (right). Separated by thin lines or spacing.
- **Disclaimer Zone (10%):** Small fine-print disclaimer at bottom

## Copy Structure

### Bundle Name Formula
> "[OUTCOME] BUNDLE" or "[OUTCOME] STACK" or "THE [OUTCOME] SYSTEM"

### Benefit Formula
> [Emoji] [Outcome-driven benefit in 3-5 words]

### Placeholders
- `[BUNDLE_NAME]` — Name of the bundle (STAY SHARP BUNDLE, GUT RESET STACK)
- `[DISCOUNT]` — Percentage off (25%, 40%, 47%)
- `[PRODUCT_1-3]` — Individual product names
- `[EMOJI_1-3]` — Relevant emoji for each benefit
- `[BENEFIT_1-3]` — Short outcome-driven benefit per product
- `[OFFER]` — Discount or bundle deal

## Image Generation Prompt

```
A clean, modern supplement ad on a [soft lavender/sage green/warm beige] background. 
At the top-left, a [purple/green/coral] rounded badge reads "[DISCOUNT]% OFF". 
Below, a bold sans-serif headline: "[BUNDLE NAME]" in [dark purple/navy/forest green].

The center contains three horizontal product rows, evenly spaced:
- Row 1: A [supplement bottle/packet] product photo on the left, with "[emoji] [Benefit text]" 
  in medium-weight text to the right
- Row 2: Same layout, different product
- Row 3: Same layout, different product

Each product image is a clean cutout on the background color. The benefit text uses 
[matching brand color] for the emoji and dark text for the benefit statement.

At the bottom, small disclaimer text in light gray. Clean, modern, scannable layout. 
No clutter. Aspect ratio 4:5.
```

## Best For
- Supplement bundles/stacks
- Multi-product offers
- Subscription box introductions
- Health systems (gut health, cognitive, hormonal)
- Any product line with 3+ complementary products

## Design Tips
- Emojis are CRITICAL — they create visual anchors and break up text for fast scanning
- Keep benefits to 3-5 words max (outcome, not ingredient)
- The discount badge should be the first thing the eye hits
- Soft, muted backgrounds (lavender, sage) work better than white for supplements
- Each row should feel like a "mini-ad" that works on its own
- The bundle concept reduces decision fatigue — "just get all three"
