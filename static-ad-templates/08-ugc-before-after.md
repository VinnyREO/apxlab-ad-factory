# Template 08: UGC Before/After Transformation

## Description
Side-by-side or sequential UGC-style photos showing transformation. Looks like a real person's social media post, not a polished ad. Works through aspiration and social proof simultaneously.

## Visual Layout

### Option A: Side-by-Side Split
```
┌──────────────┬──────────────┐
│              │              │
│   BEFORE     │   AFTER      │
│              │              │
│  (dimmer,    │  (brighter,  │
│   less       │   more       │
│   flattering)│   flattering)│
│              │              │
│              │              │
└──────────────┴──────────────┘
```

### Option B: Same Frame Comparison
```
┌─────────────────────────────┐
│                             │
│   [SINGLE UGC PHOTO]        │
│   Person in casual setting  │
│   Showing result visually   │
│   (fit, skin glow, energy)  │
│                             │
│   NO TEXT OVERLAY            │
│   (all copy in ad body)     │
│                             │
└─────────────────────────────┘
```

### Zones
- **Image Zone (100%):** Full-frame UGC photo(s). No text overlays, no branding, no filters.
- All persuasion copy lives in the Meta ad body text, not on the image.

## Copy Structure

### Ad Body Formula
> [Hook — personal story opening]
> [Pain point narrative — what they struggled with]
> [Discovery moment — how they found the product]
> [Transformation result — specific, measurable]
> [Emotional payoff — how life changed]
> [CTA]

### Placeholders
- `[TRANSFORMATION_TYPE]` — Weight loss, skin clarity, hair growth, energy, fitness
- `[BEFORE_STATE]` — Specific description of the problem
- `[AFTER_STATE]` — Specific description of the result
- `[TIMEFRAME]` — How long the transformation took
- `[FAILED_ALTERNATIVES]` — What they tried before that didn't work

## Image Generation Prompt

```
Option A - Side-by-Side:
Two casual mirror selfie photos placed side by side. Left ("before"): person in 
[casual clothes], slightly dimmer lighting, less confident posture. Right ("after"): 
same person, same mirror, same outfit style, but brighter lighting, more confident posture, 
[visible transformation]. Both photos look like authentic iPhone photos — not studio shots. 
Bathroom/bedroom mirror setting. Aspect ratio 4:5.

Option B - Single Transformation:
A casual, candid photo of a [person] in [everyday setting] showing [transformation result]. 
Natural lighting, iPhone-quality, no filters. The person looks genuinely happy/confident. 
Could be: trying on clothes, at the beach, cooking in kitchen, at gym. 
NO text, NO branding. Looks like a real Instagram/Facebook post. Aspect ratio 4:5.

Option C - Lifestyle Moment:
A warm, candid photo of [person] with [family/friends] in a natural setting. Everyone is 
smiling. The subject looks healthy, energetic, and present. The image implies transformation 
without showing "before" — the happiness IS the after. Aspect ratio 4:5.
```

## Real Examples from Research

- **Emma Grey** — "No more fighting your own body...👉" with side-by-side mirror selfie. Long-form personal journey copy about menopause weight gain. Score: 61. 6 image variants.
- **Savvy Housekeeper** — Before/after comparison (product transformation). Score: 91.
- **Ageless Beauty Secrets** — "My daughter said, 'Mom, you're smiling...'" emotional third-party observation. Score: 41.
- **WarriorBabe** — "(Free Quiz) Discover What Your Body Needs to Change After Menopause." Score: 41. Quiz funnel entry point.

## Best For
- Weight loss / body transformation products
- Skincare / anti-aging
- Hair growth products
- Fitness programs and supplements
- Any product with visible, physical results
- Products targeting women 35+ (especially menopause/aging concerns)

## Design Tips
- AUTHENTICITY IS EVERYTHING — overproduced images kill this format
- The "before" should look real, not purposely unflattering
- Same setting in both photos = more credible comparison
- NO text on the images — let the visual speak, copy does the selling
- Long-form ad copy works best with this format (story-based selling)
- The emotional hook often comes from a third person noticing the change
- Mirror selfies and bathroom settings = maximum relatability
- This format can run into Meta policy issues — avoid before/after claims that violate ad policies
