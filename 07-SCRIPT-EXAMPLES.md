# Script Examples — Annotated Winning Templates

## Template 1: 30-Second PAS (Problem-Agitate-Solve)

```
[0-3s] HOOK: "[Relatable pain point as question or bold statement]"
[3-8s] AGITATE: "I tried [alt 1], [alt 2], even [alt 3] — nothing worked / 
        I was still [suffering from problem]"
[8-18s] SOLUTION: "Then I discovered [Brand]. It has [key ingredient] that 
         [specific mechanism of action]"
[18-25s] PROOF: "After [timeframe], I noticed [specific result]. 
          [Optional: social proof stat]"
[25-30s] CTA: "Get yours at [website] — they're doing [offer] right now. 
          Link in bio"
```

**Why it works:** Names failed alternatives (builds credibility), specific mechanism (not just "it works"), timeframed results (believable), urgency CTA.

## Template 2: 60-Second Testimonial

```
[0-3s] HOOK: "I was skeptical about [product/category] but here's what happened..."
[3-10s] BACKSTORY: "For [timeframe], I struggled with [problem]. 
         I tried [failed solution 1], [failed solution 2]..."
[10-20s] DISCOVERY: "My [credible source — doctor/friend/trainer] 
          recommended [Brand]. What's different is [key differentiator]"
[20-35s] EXPERIENCE: "The first thing I noticed was [immediate benefit]. 
          Then after [timeframe], [deeper benefit]. 
          I also love that [secondary benefit/feature]"
[35-50s] SOCIAL PROOF: "I'm not the only one — they have [X reviews/awards]. 
          [Optional: show review on green screen]"
[50-60s] CTA: "If you're dealing with [problem], seriously try [Brand]. 
          They have a [guarantee/offer]. Link in my bio."
```

**Why it works:** Skeptic-to-believer arc mirrors viewer's mindset. Credible referral source. Layered proof (personal + social). Risk reversal at CTA.

## Template 3: "X Reasons Why" (Quick Hits)

```
[0-3s] HOOK: "[Number] reasons I switched to [Brand] for my [need]"
[3-8s] REASON 1: "First, [strongest/most unique benefit with specifics]"
[8-15s] REASON 2: "Second, [addresses common objection or pain point]"
[15-22s] REASON 3: "Third, [social proof element or unexpected benefit]"
[22-28s] SUMMARY: "Honestly, I wish I found this sooner"
[28-30s] CTA: "Link below — they're running [offer] right now"
```

**Why it works:** Listicle format creates curiosity gap. Each reason addresses a different concern. Fast-paced = high retention.

## Template 7: Buyer Decision Questions (BDQ)

```
[0-3s] HOOK: "Answering the top 3 questions about [Brand/Product]"
[3-10s] Q1: "'Does it actually work?' — [Answer with proof/specifics]"
[10-18s] Q2: "'What does it taste like/feel like?' — [Honest, relatable answer]"
[18-25s] Q3: "'Is it worth the price?' — [Value reframe]"
[25-30s] CTA: "Try it risk-free — [offer/guarantee]"
```

## Essential Script Elements Checklist
Every script should include at least 4 of these 6:
- [ ] **Hook** (first 2-3 seconds — visual + audio + text)
- [ ] **Problem identification** (relatable pain point)
- [ ] **Solution introduction** (product as the answer)
- [ ] **Value propositions / USPs** (what sets it apart)
- [ ] **Social proof** (reviews, awards, press, testimonials)
- [ ] **Call to action** (with urgency)
