# Avatar Archetypes for Supplement UGC

## ⚡ STRATEGIC NOTE: ICPs > Brand Ambassadors

Avatars should be **ICP mirrors**, not brand ambassadors. The person in the ad represents the BUYER, not the brand. Viewers convert when they think "that's literally me."

**For each brand:** Start with these generic archetypes, then customize into pain-point-specific ICPs based on the brand's actual customer segments. See `data/brands/huel-avatars.md` for a worked example of the ICP approach.

Based on industry research — each brand should have 3-5 ICPs in rotation to cover audience segments and avoid "spokesperson fatigue."

---

## 🧬 The Biohacker / Optimizer
**Best for:** Performance supplements, protein, nootropics, pre-workout
**Brands:** AG1, Onnit, 1st Phorm, Ghost
**Appearance:** Lean, tech-savvy, early 30s
**Gender:** Both (but male-skewing for performance)
**Settings:** High-tech home office, modern gym, minimalist apartment
**Clothing:** Athletic wear, fitted tee, smart casual
**Key trait:** Focus, efficacy, optimization
**Prompt snippet template:**
```
Athletic [man/woman], early 30s, lean muscular build, clean-shaven/natural look, wearing fitted performance wear in [dark/neutral] tones, sharp focused expression, tech-forward vibe
```

## 🌿 The Nurturing Expert
**Best for:** Greens powders, daily vitamins, family supplements, gut health
**Brands:** LyfeFuel, Ritual, Seed, Ka'Chava
**Appearance:** Approachable, warm, late 30s-40s
**Gender:** Female-skewing (but works for both)
**Settings:** Bright naturalist kitchen, garden, farmers market
**Clothing:** Casual earth tones, linen, sage/cream
**Key trait:** Trust, safety, knowledge
**Prompt snippet template:**
```
Warm approachable [woman/man], late 30s, natural beauty, gentle smile, wearing casual earth-tone clothing like [linen top/sage cardigan], radiating calm competence and wellness knowledge
```

## ✨ The Beauty-First Influencer
**Best for:** Collagen, gummies, skin-health, beauty supplements
**Brands:** Goli, Olly, Vital Proteins, HUM Nutrition
**Appearance:** Radiant skin, "natural" makeup look, mid-20s
**Gender:** Primarily female
**Settings:** Bathroom vanity, bedroom, bright selfie spots
**Clothing:** Pajamas, robe, casual chic, pastels
**Key trait:** Aesthetic results, flavor, glow-up
**Prompt snippet template:**
```
Radiant young woman, mid-20s, glowing clear skin, minimal natural makeup, [hair description], wearing [silk pajamas/casual chic outfit] in soft [pink/lavender/cream] tones, effortlessly photogenic
```

## 👩‍👧 The Busy Parent
**Best for:** Stress relief, sleep gummies, multivitamins, meal replacements
**Brands:** SmartyPants, Huel, Nature Made, Garden of Life
**Appearance:** Relatable, slightly fatigued but energetic, 30s
**Gender:** Both
**Settings:** Kitchen (slightly messy), car, living room, morning chaos
**Clothing:** Loungewear, casual everyday, athleisure
**Key trait:** Convenience, "daily survival," real life
**Prompt snippet template:**
```
Relatable [mom/dad], early-mid 30s, natural look with slight under-eye fatigue but warm genuine energy, wearing comfortable [loungewear/casual clothes], multitasking vibe, authentic not polished
```

## 💪 The Transformation Story
**Best for:** Weight management, protein, fitness supplements
**Brands:** Alani Nu, 1st Phorm, Bloom, Transparent Labs
**Appearance:** Fit but approachable (not bodybuilder), late 20s
**Gender:** Both (female-skewing for wellness, male for performance)
**Settings:** Gym, kitchen meal prep, mirror selfie, outdoor run
**Clothing:** Sports bra/tank, leggings, gym wear
**Key trait:** Progress, before/after energy, dedication
**Prompt snippet template:**
```
Fit [woman/man], late 20s, toned athletic build (not bodybuilder), [skin tone], wearing [colorful gym outfit/sports bra and leggings], confident post-workout glow, showing dedication without intimidation
```

## Matching Archetypes to Brand Categories

| Brand Category | Primary Archetype | Secondary | Why |
|---|---|---|---|
| Greens/Daily | Nurturing Expert | Biohacker | Trust + optimization |
| Protein/Performance | Biohacker | Transformation | Focus + results |
| Gummies/Beauty | Beauty Influencer | Busy Parent | Aesthetic + convenience |
| Sleep/Stress | Busy Parent | Nurturing Expert | Relatable + trust |
| Meal Replacement | Busy Parent | Transformation | Convenience + health |
| Nootropics | Biohacker | Nurturing Expert | Focus + knowledge |

## Scene × Archetype × Funnel Position

| Funnel Stage | Best Archetype | Best Scene | Format |
|---|---|---|---|
| Hook (0-3s) | Any — matches brand | POV selfie, car, street | Pattern interrupt |
| Problem (3-10s) | Busy Parent / Transformation | Messy desk, tired morning | Pain point |
| Solution (10-20s) | Nurturing Expert / Biohacker | Kitchen, prep routine | Product showcase |
| Proof (20-30s) | Transformation / Beauty | Mirror, before/after | Testimonial |
| CTA (30s+) | Any | Close-up product hero | Direct action |

## Creating Avatars from Archetypes

When onboarding a new brand:
1. Identify brand category → pick primary + secondary archetype
2. Customize the prompt snippet with specific details (hair, skin tone, age, clothing that matches brand colors)
3. Generate 2-3 test images to lock in the look
4. Save as avatar in Media Bank with `promptSnippet` and reference images
5. Use consistently across all generations for that brand
