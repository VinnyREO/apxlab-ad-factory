# APXlab Ad Factory — Knowledge Base

This repository contains the knowledge files for the **Ad Factory** Claude Project by [APXlab](https://apxlab.com).

## What is this?

A structured knowledge base that powers an AI-driven ad creation system. These files are uploaded as **Project Knowledge** in Claude Projects, enabling Claude to generate high-converting direct-response video ad scripts, storyboards, and image prompts.

## Files

| File | Description |
|------|-------------|
| `02-AD-FRAMEWORKS.md` | Core advertising frameworks (PAS, AIDA, BAB, etc.) |
| `03-HOOK-FORMULAS.md` | Proven hook formulas for stopping the scroll |
| `04-SCENE-LIBRARY.md` | Visual scene templates for storyboards |
| `05-AVATAR-ARCHETYPES.md` | Customer avatar definitions and targeting |
| `06-STORYBOARD-TEMPLATE.md` | Storyboard structure and formatting |
| `07-SCRIPT-EXAMPLES.md` | Example scripts across niches |
| `08-IMAGE-PROMPT-GUIDE.md` | AI image generation prompt engineering |
| `09-PRODUCTION-CHECKLIST.md` | Pre-publish quality checklist |
| `10-COMPLIANCE-REFERENCE.md` | Ad platform compliance guidelines |

### Prompts

The `prompts/` folder contains saved prompt templates for specific client niches.

## Usage

1. Create a new **Claude Project**
2. Upload these files as **Project Knowledge**
3. Set the system prompt separately (not included here)
4. Start generating ads!
