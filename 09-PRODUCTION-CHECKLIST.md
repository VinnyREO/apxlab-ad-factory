# Production Checklist — From Brief to Final Video

## Phase 1: Brief & Strategy (5 min)

- [ ] Paste brand brief into this project
- [ ] Review 3 generated concepts
- [ ] Pick 1-2 concepts to produce
- [ ] Confirm avatar, framework, duration

## Phase 2: Asset Generation (20-30 min)

### Images (Midjourney / Flux / Imagen)
From the storyboard, generate in this order:

1. **Avatar hero shot** — Used across multiple frames. Get this right first.
   - Generate 4 variations, pick the best
   - Save the exact prompt that worked — reuse for all frames with this avatar

2. **Product shots** — Lifestyle and clean versions
   - 1x lifestyle (in-scene with avatar or setting)
   - 1x clean product shot (for CTA end frame)

3. **Scene frames** — Each unique frame from the storyboard
   - Copy avatar description exactly from hero shot prompt
   - Change only: setting, pose, expression, action
   - Generate 2 variations per frame, pick best

4. **CTA end frame** — Product + space for text overlay

### Voiceover (ElevenLabs)
- [ ] Copy the full VO script from the storyboard
- [ ] Select voice matching the avatar (age, gender, energy level)
- [ ] Generate with natural pacing — slightly conversational, not announcer
- [ ] Download MP3

### Text/Captions
- [ ] Note all text overlays from storyboard (hook text, key points, CTA)
- [ ] Pick caption style: bold white with black outline (default), or yellow highlight

## Phase 3: Assembly (15-20 min)

### In CapCut or Video Editor:
1. Import all generated images in frame order
2. Set each image duration per storyboard timing
3. Add Ken Burns effect (subtle zoom/pan) to each still — prevents static feel
4. Layer voiceover audio, align to frames
5. Add text overlays per frame spec
6. Add auto-captions → adjust styling and timing
7. Add background music (low volume, mood-matching)
8. Add transitions between frames (hard cuts mostly, occasional swipe)
9. Ensure CTA frame has product visible + offer text

### Quality Check:
- [ ] Total duration matches spec (15s/30s/60s)
- [ ] Hook grabs attention in first 2 seconds
- [ ] Text is readable on mobile (not too small)
- [ ] Audio levels balanced (VO clear over music)
- [ ] Captions accurate and timed correctly
- [ ] CTA is clear (verbal + visual + caption)
- [ ] Exported 9:16, 1080x1920, MP4

## Phase 4: Variations (10 min)

- [ ] Create 2 additional versions with different hooks (swap first 3 seconds)
- [ ] Each hook gets its own export
- [ ] Label files: `[brand]-[concept]-hookA.mp4`, `hookB.mp4`, `hookC.mp4`

## Phase 5: Delivery

### Files to deliver per ad concept:
```
📁 [Brand]-[Concept]/
├── hookA.mp4          (version with hook variation A)
├── hookB.mp4          (version with hook variation B) 
├── hookC.mp4          (version with hook variation C)
├── script.md          (full script + storyboard)
├── assets/
│   ├── avatar-hero.png
│   ├── frame-1-hook.png
│   ├── frame-2-problem.png
│   ├── frame-3-solution.png
│   ├── frame-4-proof.png
│   ├── frame-5-cta.png
│   ├── product-lifestyle.png
│   └── product-clean.png
└── voiceover.mp3
```

## Time Estimate Per Ad

| Phase | Time |
|-------|------|
| Brief & strategy | 5 min |
| Image generation | 20-30 min |
| Voiceover | 5 min |
| Assembly | 15-20 min |
| Variations | 10 min |
| **Total** | **~60 min per ad concept** |

With 3 hook variations per concept, that's **3 testable ads per hour**.

## Compliance Quick Check (Before Publishing)

- [ ] No disease/cure claims ("cures," "treats," "heals," "fixes")
- [ ] No guaranteed results ("will make you," "100% effective")
- [ ] No negative self-perception ("tired of looking/feeling [negative]?")
- [ ] Uses structure/function language ("supports," "promotes," "helps maintain")
- [ ] FTC disclosure if paid partnership
- [ ] No misleading before/after images
